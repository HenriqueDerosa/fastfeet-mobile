# Fastfeet Mobile

## Getting started

1. use `yarn` to download dependencies
2. run it in one of the following devices

| ios     | `yarn ios`     |
| ------- | -------------- |
| android | `yarn android` |

You can either use genymotion, ios simulator in your mac, or even an actual device.
